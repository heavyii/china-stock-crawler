# china-stock-crawler

抓取中国股市数据。

数据来源是新浪财经的API。

当前仅支持在命令行抓取个股的基本信息、股本结构、财务状况、行情数据。

详细的说明参见[Wiki](https://github.com/squirrelm/china-stock-crawler/wiki/index)。

另外，原本是只想取基本信息和行情数据的，但由于必须要股本结构和财务状况中的数据方可计算行情数据的部分字段，所以将它们也都列出来。

# 使用方法
命令行执行，例如：
```
php cli\get_single_stock.php sh601998
```
